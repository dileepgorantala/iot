LED-Light Display

contains driver for 8x8 LED Modules controlled via MAX722xx. It allows you to build display of custom size that is only limited by the hardware itself. .

Obstacle avoiding robot Car

I will teach you how to make an obstacle avoiding robot that works with Arduino. You must be familiar with Arduino. .

Bluetooth fan and light control

we will control a small fan by bluetooth using the Arduino board and a smartphone.This is why we will create two programs: a mobile application with App Inventor for the smartphone and a program for the Arduino board..

Fingerprint Door Lock

The biometric fingerprint sensor is ideal to create a system capable of protecting what you require through the analysis of your fingerprint. The device works with the serial protocol, so it can be used with any microcontroller (arduino etc..).

Control LED By Clap

If you clap two times, the lamp will toggle its output on or off. If you clap three times, the lamp will fade up and down until you clap a fourth time to stop it..

Water Plant

In this guide, learn to connect the kit to the Arduino or as a standalone, then program it to monitor soil moisture levels. When the soil moisture level goes below the set threshold, the plant will be automatically watered..

LED-Light Display

Here is the list of components required to build this project.

  • Arduino UNO
  • 8x8 LED Matrix with MAX7219
  • Breadboard
  • Header Wires
  • 5V power supply
  • Arduino IDE

Obstacle avoiding robot Car

Here is the list of components required to build this project.

  • Arduino Uno
  • Ultrasonic HC – SR04 sensor.
  • Motor Driver IC – L293D.
  • Servo Motor (Tower Pro SG90)
  • Geared Motors x 2.
  • Robot Chassis.
  • 18650 Battery.
  • battery Holders.

Bluetooth fan and light control

Here is the list of components required to build this project.

  • Arduino UNO
  • HC-05 Bluetooth Module
  • 1838 IR Receiver IC (with metal case)
  • 2k Resistor 1/4 watt
  • 4.7k Resistor 1/4 watt
  • Pushbuttons or switches
  • 4-channel Relay Module 5V

Fingerprint Door Lock

Here is the list of components required to build this project.

  • DFRobot Fingerprint Sensor
  • Arduino UNO
  • 12V 5A Power Supply

Controled by LED Clap

Here is the list of components required to build this project.

  • Solderless Breadboard
  • Arduino UNO,
  • Sound Sensor,
  • 1-Channel 5v Relay Module,
  • Male to Female jumper Wires,
  • Male to Male jumper Wires,
  • Bulb Holder,
  • 220v LED Bulb,
  • 5v 2Amp Power Adapter
